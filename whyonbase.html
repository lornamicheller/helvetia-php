¿POR QUÉ
ONBASE?

OnBase es una excelente solución que hace parte de una categoría de software cuya importancia, en el medio de los negocios, es mayor cada día y es conocida como Gestión Integrada de Documentos o IDM (Integrated Document Management). Los sistemas IDM permiten administrar prácticamente cualquier tipo de documento, como por ejemplo, imágenes, reportes, archivos de aplicación, formas HTML, mensajes de correo electrónico, archivos de video y audio etc, así como también, cada etapa del ciclo de vida de un documento, desde la creación, almacenamiento, recuperación y revisión hasta la distribución.

Otros sistemas tienen un enfoque de ''kit de herramientas'' en el que proporcionan una serie de ''componentes'' con una interfase común que interactúa con diversos paquetes de software o soluciones. OnBase, por el contrario, tiene la ventaja de ser una única aplicación de software que utiliza una sola base de datos SQL, una sola interfase de configuración y una única interfase de usuario para la recuperación y procesamiento de documentos. Este nivel de integración sin precedentes permite que OnBase soporte, sin necesidad de programación, las más sofisticadas soluciones de administración de documentos y flujos de trabajo (Workflow).

OnBase esta desarrollada con tecnología de avanzada, bajo una arquitectura cliente / servidor, habilitada y basada en Web, que aprovecha la transparencia y la escalabilidad de bases de datos como Oracle y Microsoft SQL Server para proporcionar soluciones de alto desempeño y cubrir desde soluciones departamentales hasta soluciones corporativas. OnBase ofrece al usuario la flexibilidad de tener acceso al sistema a través de diferentes clientes WEB (es decir navegadores de Internet), clientes grueso (la interfase propia de OnBase para Windows), del Windows Explorer® y mejor aun desde otras aplicaciones.

OnBase es configurable en su totalidad por medio de herramientas Windows de ''apunte y clic'', lo que permite la rápida implantación de soluciones sofisticadas sin necesidad de una programación costosa y demorada. Para diseñar soluciones que cubran los requerimientos específicos de cada empresa, los usuarios pueden elegir de entre más de 30 módulos OnBase, que se licencian por separado y proporcionan funcionalidades de entrada, administración y salida al sistema. Por esta razón, OnBase incentiva a que sus clientes inviertan hoy en la solución especifica para su problema actual y en el futuro, amplíen gradualmente el sistema de acuerdo con sus necesidades y con una relación costo-beneficio efectiva.
